# Code Signing Certificate Manager

A full-stack application for creating Digital Certificates and signing files with SHA-256 encryption.  
This system simulates enterprise code-signing workflows used in secure software delivery pipelines.

# Features

 Generate RSA Digital Certificates  
 Store Certificates in MongoDB  
 Upload & Sign Files (SHA-256)  
 View Signed Files & Certificates  
 Full Stack App (Spring Boot + Angular)  
 REST API Architecture
